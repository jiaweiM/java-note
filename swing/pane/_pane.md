# 容器

## 简介

| 容器           | 说明                   |
| -------------- | ---------------------- |
| [JPanel](./JPanel.md)   | 常规容器               |
| JScrollPane    | 支持滚动条             |
| JSplitPane     | 分隔成两部分           |
| JTabbedPane    | 选项卡                 |
| JInternalFrame | 提供轻量级本机窗口功能 |
| JLayeredPane   | 允许组件重叠           |
