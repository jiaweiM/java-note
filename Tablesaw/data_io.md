# 导入和导出数据

## 支持的格式

| Format                         | Import | Export |
| :----------------------------- | :----- | :----- |
| CSV (and other delimited text) | Yes    | Yes    |
| JSON                           | Yes    | Yes    |
| RDBMS (via JDBC)               | Yes    |        |
| Fixed Width Text               | Yes    | Yes    |
| Excel                          | Yes    |        |
| HTML                           | Yes    | Yes    |
