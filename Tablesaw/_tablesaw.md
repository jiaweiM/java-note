# Tablesaw

Tablesaw 包含表格处理工具、创建统计模型和可视化功能。换言之，它就是一个带有附加功能的 dataframe。

dataframe 是内存中的表格。Tablesaw 提供的操作：

- 从文本或数据库导入和到处数据
- 添加和删除 columns
- 排序
- 过滤
- 通过向已有 column 应用函数来创建新的 column
- column 或 table 汇总
- 通过 appending 或 joining 合并表格
- 计算统计信息
- 添加、更新和删除 rows

另外提供可视化功能。
